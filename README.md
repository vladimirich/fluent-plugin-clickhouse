# Fluent::Plugin::Clickhouse, a plugin for [Fluentd](http://fluentd.org)

Send your logs to Clickhouse

```ruby
gem 'fluent-plugin-clickhouse'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install fluent-plugin-clickhouse
